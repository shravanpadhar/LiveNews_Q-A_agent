# 🤖 Live News & Q&A AI Agent

An intelligent, autonomous AI Agent built with **JavaScript**, **LangChain**, and **LangGraph**. This agent connects to Google's **Gemini 2.5 Flash** model and uses **SerpAPI** to browse the web for live, up-to-date news and information before answering user queries.

---

## ✨ Features
- **Real-Time Search:** Uses Google Search capabilities to bypass LLM training data cutoffs.
- **Autonomous Planning:** Uses the ReAct agent framework to decide *when* to search vs. *when* to answer immediately.
- **Modern Architecture:** Built using the latest modern LangGraph standard (`createReactAgent`).
